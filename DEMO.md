## Demo Overview
This video demonstrates the core functionality of the Thought-to-Task Agent.
The agent converts unstructured human thoughts into structured, prioritized tasks
and generates a simple daily execution plan.

## Notes
Due to free-tier quota limitations, the demo runs in a fallback mode.

without changing the frontend or API contract.
